type MyFSStMonad m a = StateT FilePath m a Source #

Every operation is completed and exceptions handled within a context of parameter m in this type alias, and StateT handles preserving of the current filepath between iterations of interactive REPL.

listProcessors :: MyFSMonad m => Map String (FilePath -> MyFSStMonad m ()) Source #

A map that stores functions that process commands from the recognizable list. Commands being:

  • ls - processLS
  • help - processHelp
  • mkdir - processMkdir
  • touch - processTouch
  • cat - processCat
  • echo - processEcho
  • rmdirectory - processRemoveDirectory
  • rmfile - processRemoveFile
  • locate - processLocate
  • fileinfo - processFileinfo
  • dirinfo - processDirinfo
